## 검증 계획

### 수동 검증
1. 스케줄러를 백그라운드로 실행한 후 약 2~3분간 로그와 생성되는 CSV 파일을 관찰합니다.
2. 생성된 CSV 파일의 데이터 구조(헤더, 데이터 행, 인코딩 등)가 정상인지 확인합니다.
3. 1분 주기로 정상적으로 파일이 갱신되거나 누적되는지 확인합니다.
